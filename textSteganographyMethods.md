There are various methods for hiding text inside a text carrier. Below, are the common one.

# Common Text Steganography Methods

 - ## Using the sentence features in the carrier texts
	 - In this approach, one would take advantage of commas, periods, and other punctuations to represent the bits in the Ascii code of the alphabet letters, like "," for 0 and "." for 1. This method is most useful when the carrier text is large enough with multiple use of punctuation characters.
	 - Another approach would be to add a meaningful sign like "!" next to the world that contains a letter of the secret message. In this approach, some sort of agreement between the sender and the receiver must be in place.
 - ## Using the structure of the carrier text document
	 - In this approach, one would take advantage of spacing with tabs and single spaces conveying different values for the bits of the Ascii characters from the hidden message.
	 - The same can be also applied for line spacing with paragraph spacing to consume slightly different vertical spaces than normal but invisible to notice. Each paragraph spacing then represent the Ascii bits of the hidden message text.
 - ## Taking advantage of semantic modications
	 - In this approach, one would replace highly frequent words in the carrier text with their synonyms that contain characters or words of the hidden message.
	 - In another approach, one would take advantage of numeric replacement to represent numbers in the carrier text. For instance, instead of the word "seventy", the text can include the numebr "70" instead where "7", "0", or both "70" letters can convey the letters from the hidden message.
 - ## Using specific file formats
	 - In this approach, specific files like HTTP files that contain tags and links to different page objects can be used to create a stegano. For instance, URL manipulation or CSS properties can be used for this purpose.
	 - Another approach would be to use programming source codes that include convulted codes. In this approach, one would hide the letters of a message in various places inside the source code such as "print" instructions or functions variable names.
	 
## New: AI based methods
Using generative AI is common and recent approach where one would use AI to hide secret message letters inside a generated text. This may also include using patterns for the occurance of the secret message letters.
